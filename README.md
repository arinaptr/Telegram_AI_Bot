# Telegram AI Bot

![Bot Logo](/assets/logo.png)

## Описание

Telegram AI Bot - это многофункциональный бот для Telegram, способный выполнять различные задачи обработки текстовых, аудио- и видео-сообщений, а также изображений. Бот использует внешние инструменты и сервисы для выполнения специфических задач, таких как преобразование текста в речь, транскрипция аудио и видео, извлечение краткого содержания текста и многое другое.

## Функциональные возможности

- Обработка текстовых сообщений
- Обработка ссылок на YouTube видео
- Обработка аудио- и видео-сообщений
- Обработка изображений
- Ответы на различные команды
- Преобразование текста в речь
- Преобразование текста в изображения
- Транскрипция аудио и видео
- Получение краткого содержания текста

Для использования Telegram AI Bot потребуется установить следующие инструменты и зависимости:

- [FFmpeg](https://ffmpeg.org/)
- [VoiceRSS](https://www.voicerss.org/)
- [Stable Diffusion](https://stablediffusionweb.com/)
- [OpenAI GPT](https://openai.com/)
- [OpenAI Whisper](https://openai.com/research/whisper)
- [IronOCR](https://ironsoftware.com/)
- [YouTube DLP](https://github.com/yt-dlp/yt-dlp)
- [Detect Language](https://detectlanguage.com/)
- [Telegram](https://core.telegram.org/bots/api)
